## 📂 Klasör Oluşturma

* `Create New File` butonuna tıklayın
* Resimdeki alana `/` koyma durumunuzda klasör açılır
* Açılan klasörün altına `README.md` dosyası ekleyin
* Dosyanın içine `# Dizin adı` yazın
* `Commit New File` butonuna basın
