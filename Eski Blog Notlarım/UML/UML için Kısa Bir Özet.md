# UML için Kısa Bir Özet

```uml
[Erişilebilirlik] [Değişken İsmi] : [Değişken Türü]
[Erişilebilirlik] [Fonksiyon İsmi] ( [Parametrelerin Türü] ) : [Return Tipi]
```

```uml
#deneme : int
+fonksiyon(int, int) : void
-fonk2() : String
```

## Erişilebilirlik

- `+`  Public
- `-`  Private
- `#` Protected
- `~` Package

## Tür

- void
- int
- char
- String
- ...
